# AI Smart Interaction System

## Overview
An AI-powered desktop interaction system that uses computer vision, gesture recognition, emotion detection, and automation.

## Features
- Hand Gesture Recognition
- Emotion Detection
- Desktop Automation
- Analytics Dashboard
- Flask Web Interface
- SQLite Database Logging

## Technologies Used
- Python
- Flask
- OpenCV
- MediaPipe
- TensorFlow
- HTML/CSS/JavaScript
- SQLite

## Gestures

| Fingers | Action |
|----------|---------|
| 1 | Calculator |
| 2 | File Explorer |
| 3 | Calendar |
| 4 | Terminal |
| 5 | Settings |

## Installation

```bash
pip install -r requirements.txt
python app.py
```
